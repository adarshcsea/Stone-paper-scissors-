# 🎮 Stone Paper Scissors

A modern browser-based implementation of the classic **Stone Paper Scissors** game built using HTML, CSS, and JavaScript.

Play against the computer with smooth animations, an interactive UI, and real-time score tracking.

## ✨ Features

- Classic Stone Paper Scissors gameplay
- Modern responsive interface
- Computer AI opponent
- Real-time score tracking
- Instant winner announcement
- Smooth animations
- Mobile-friendly design
- Lightweight and fast

---

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

---

## 🎯 Game Rules

- Stone beats Scissors
- Scissors beats Paper
- Paper beats Stone
- Same choices result in a Draw

---

## 🚀 How to Run

Clone the repository

```bash
git clone https://github.com/adarshcsea/Stone-paper-scissors-.git
```

Open `index.html` in your browser
